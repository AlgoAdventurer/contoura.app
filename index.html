Built for the full orienteering rhythm.

The first version is designed to close the gaps between training logs, race calendars, maps, result lists and personal performance analysis.

T

Log training

Capture distance, climb, map, location, goals, actions, learnings, attachments and Livelox links in a structured way.

R

Prepare races

Keep fixture details, start lists, start time, strategy, resources and results together in one race record.

A

Analyze deeper

Compare ratings, trends, mistake density, start loss, control types, race quarters and performance highlights.

P

Control planning

Build a training calendar with categories, intensity, weekly targets, monthly targets and coach suggestions.

D

Connect data

Treat o-l.ch fixtures, start lists, result lists, IOF XML and Livelox as central sources for orienteering-specific data.

S

Make statistics visible

Use training, racing, performance, favorite cards and detail views to quickly see where development is happening.

From planning to reflection.

Contoura is not meant to be another loose logbook. It is designed around the weekly workflow ambitious orienteers already follow.

1

Plan

Set the training week, load, technical focus and race goals before the work starts.

2

Collect

Keep maps, links, results, splits, files and notes in the right place.

3

Evaluate

Compare technique, mental state, running feel, preparation and overall impression.

4

Learn

Recognize patterns and shape the next session from real insight.

"After every orienteering race I know something. Contoura should make sure I can still use it six weeks later."

Product principle for the first validation phase
Ambitious juniors More structure for training, reflection and season development.
Elite and performance athletes Clean data for race comparison, mistake analysis and preparation.
Coaches, clubs and squads Better visibility of suggestions, evaluation backlog and progress.
